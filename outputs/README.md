# Outputs Klasörü

Analiz sonuçları bu klasöre kaydedilir.

- `rfm_segments.csv` - RFM analizi sonuçları
- `cltv_prediction.csv` - CLTV tahminleri

- `yeni_marka_hedef_musteri_id.csv` - Özel hedef kitle (RFM case 1)
- `indirim_hedef_musteri_ids.csv` - İndirim hedef kitle (RFM case 2)
